## Quick start

**Use `yarn` instead of `npm` to manage dependencies**

**prerequisites**

- Have Node.js v6+, and [Yarn](https://yarnpkg.com/en/) installed

1. clone repo
2. navigate to /app folder
3. run `yarn install`
4. run `yarn start`
